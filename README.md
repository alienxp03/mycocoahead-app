##MYCocoaHeads iOS app

A community iOS app. List and find all tech events in Malaysia!


##Features
- Upcoming events
- Past events
- Event list based on date and location
- About

- Actual development of the app

##Technical Details
- Supports iOS8 and above
- Supports phone only (for now)
